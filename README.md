# lab-11
Lab DescriptionThis project applies the K-Means clustering algorithm to segment a credit card company's customer base using the CC_GENERAL.csv dataset. By analyzing historical usage metrics (such as spending behavior, cash advances, balances, and payment frequencies), the algorithm groups similar users together. This allows businesses to uncover hidden customer patterns and design optimized, target-specific marketing strategies.Short Answers to Final Questions1. Why is this an unsupervised learning problem?Answer: Because the dataset does not contain predefined target labels or correct category groupings; the algorithm must find patterns entirely on its own.2. Why did we remove the CUST_ID column?Answer: It is a unique identifier string with no behavioral meaning. Keeping it would distort distance measurements during clustering.3. Which columns had missing values?Answer: MINIMUM_PAYMENTS and CREDIT_LIMIT.4. How did you handle the missing values?Answer: By using mean imputation (filling the missing slots with the average value of each respective column).5. Why is scaling important before applying K-Means?Answer: K-Means relies on Euclidean distance. Scaling ensures that high-magnitude columns (like CREDIT_LIMIT) do not mathematically overwhelm low-magnitude columns (like frequency scores).6. Which K value did you choose? Explain your answer using the elbow method and silhouette score.Answer: $K = 3$. The elbow curve shows a distinct drop deceleration (bend) starting at 3, and $K = 3$ yields the highest overall Silhouette Score (~0.251), signifying optimal cluster separation.7. Based on the cluster summary table, describe each customer segment in your own words.Answer: * Cluster 0 (Cash Borrowers): High account balances, low purchase rates, but very high dependency on cash advances.Cluster 1 (VIP Spenders): High credit limit users who purchase heavily, make frequent transactions, and maintain high payment rates.Cluster 2 (Budget Users): The largest group; they maintain very low balances, minimal spending habits, and use cards conservatively.8. Which cluster may represent high-value customers?Answer: Cluster 1, due to their dominant purchasing metrics, active transaction frequency, and high credit limits.9. Which cluster may represent customers who rely more on cash advance?Answer: Cluster 0, as they have a disproportionately large cash advance balance and transaction frequency compared to other groups.10. How can a company use these clusters for marketing strategy?Answer: Offer Cluster 0 low-fee cash advance windows or balance transfer promotions; target Cluster 1 with premium cash-back upgrades or luxury rewards; and incentivize Cluster 2 with low-interest installment offers or zero-fee benefits to increase card usage.
